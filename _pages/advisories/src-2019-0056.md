body: |
 <p class="cn" markdown="1">This vulnerability allows remote attackers to execute arbitrary code on vulnerable installations of Hewlett Packard Enterprise Intelligent Management Center. Although authentication is required to exploit this vulnerability, the existing authentication mechanism can be bypassed.</p>
 <p class="cn" markdown="1">The specific flaw exists within the handling of the renameFile function. The issue results from the lack of proper validation of a user-supplied string which can lead to arbitrary file creation. An attacker can leverage this vulnerability to execute code in the context of SYSTEM.</p>
